 New Website

    Posted on August 6, 2020 at 12:00 PM



The Robotics Club at UCF has changed name to the Robotics Club of Cental Florida. With this, the club has also gained a new website! New pages, updated old pages, and many new widget integrations await.

Out with the old and in with the new. This new website has completely overhauled the old website - every page of it is now removed and replaced. If you want to see what used to be and how the website has changed since then, you can still view the old pages here: old website
